# Antigravity CLI Opener Changelog

## [Unreleased]

## [1.0.0]

### Added

- Initial release.
- `Tools → Open Antigravity CLI` action that opens the IDE Terminal tool window,
  creates a fresh shell tab in the project root, and runs the `agr` command.
- Default keyboard shortcut `Ctrl+Alt+G` (remappable via Settings → Keymap).
- Error notification when the terminal cannot be launched.
